# Menu Marketing

Vite + React project with TypeScript, using pnpm.

## Commands

pnpm dev      # Dev server (port 3030)
pnpm build    # TypeScript check + build
pnpm check    # Lint + format
pnpm fix      # Auto-fix lint
pnpm pr_ready # CI: check + test

## Verification Required

After every change: `pnpm build && pnpm check`

## Language

ALL user-facing text in Brazilian Portuguese (pt-BR).
